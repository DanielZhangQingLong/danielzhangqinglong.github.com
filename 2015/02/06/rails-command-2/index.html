rails generate 命令生成的文件

命令 概要 controller view helper model migration
scaffold 视图到数据库全部生成 有 有 有 有 有
controller 生成 controller 和 view 有 无 有 无 无
model 生成model 无 无 无 有 有
migrate 生成移植文件 无 无 无 无 有

scaffold

rails g scaffold 可以创建 controller view model migration 所有组件, 我们可以指定 model , 其字段类型和索引, 但是不可以指定 action. 比如:

rails g controller

用于生成控制器, 我个人更偏向于使用该命令, 因为 scaffold 生成一大堆文件, 感觉乱七八糟的.

rails g model

生成 model 和

rails g migration

生成移植文件, 这是 rails 为了方便修改数据库提供的一种简单方法, 只需要写 ruby 代码就可以修改数据库, 比如创建表, 添加column 等等.
