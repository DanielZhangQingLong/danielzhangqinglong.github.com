使用Spring缩短测试文件的载入时间

发表于 2015-03-05   |   分类于 Rails   |  

Spring从Rails4.1开始已经成了”标配”, 我们在命令行敲入命令时, 由于Rails中有各种各样的library需要进行加载, 会增加命令执行的时间, 尤其是在测试的时候, 会严重影响开发效率. Spring扮演的角色是preloader-预加载器, 它事先在后台加载这些library, 以缩短敲入命令后的等待时间.
